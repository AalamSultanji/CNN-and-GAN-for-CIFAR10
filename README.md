# CNN-and-GAN-for-CIFAR10
Built and trained a custom CNN and cGAN in PyTorch on CIFAR-10, focusing on classification performance and class-conditioned image generation.
## CNN for CIFAR10:
The python file containing the architecture can be found in `convolution 1.py`, and the training and testing algorithm along with the implementation of the network can be found in `main1.py`.

##cGAN For CIFAR10: 
The code can be found in `cGAN.py`. To plot the graphs, run the code found in `plot_cgan.py` by changing the appropriate output file name.
